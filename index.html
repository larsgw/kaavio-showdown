Kaavio Showdown

A Showdown extension for enabling the use of Kaavio's manipulation API within MarkDown. The aim of this library is to allow easy creation of interactive descriptions to accompany Kaavio diagrams.

Note: Kaavio is under development. It is now only used to render diagrams in WikiPathways. Kaavio is currently being developed as Pvjs (the WikiPathways Kaavio wrapper) but it will be extracted into it's own generic library soon.
